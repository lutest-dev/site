# Lutest site

Documentation site for [Lutest](https://github.com/cayasde/lutest), built with
Astro and Starlight.

## Development

```sh
pnpm install --frozen-lockfile
pnpm dev
```

## Validation

```sh
pnpm check
pnpm build
```

The `main` branch deploys to GitHub Pages through the workflow in
`.github/workflows/deploy.yml`.
