Esta página é curta de propósito.

A documentação atual foca no runtime `lute` porque é isso que você pode usar hoje.

No longo prazo, o projeto ainda quer cobrir mais runtimes voltados para Luau.

Para Roblox, a direção atual é integrar com [Roblox Open Cloud Luau Execution](https://create.roblox.com/docs/cloud/reference/features/luau-execution), usando Luau session tasks para rodar testes em um fluxo headless no estilo do Studio, tanto para desenvolvimento quanto para CI.
